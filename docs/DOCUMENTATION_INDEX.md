## 使用建议

### 新用户
1. 阅读 README.md 了解项目
2. 查看用户手册快速上手
3. 参考 CHANGELOG 了解功能

### 开发者
1. 阅读架构设计了解系统
2. 按照开发指南搭建环境
3. 参考 API 文档进行开发

2. 参考开发指南搭建环境
3. 提交 PR 参与贡献
