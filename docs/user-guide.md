## 快速入门

### 什么是 IPFlow？

IPFlow 是一款专业的知识产权申请辅助工具，帮助您：
- 📄 自动生成软件著作权申请材料
- ✅ 智能合规性检查
- 📦 一键导出标准格式的申请文档包
- 👥 通过配套协作能力整理多个申报项目

### 首次使用流程

```
1. 注册账号 → 2. 创建/加入组织 → 3. 创建项目 → 4. 填写信息 → 5. 生成材料 → 6. 导出提交
```

### 系统要求

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **操作系统**: Windows 10+, macOS 10.15+, Linux
- **网络**: 稳定的互联网连接
- **存储**: 建议至少 1GB 可用空间用于上传代码文件

## 账号管理

### 注册账号

1. 访问 https://ipflow.com
2. 点击右上角"注册"按钮
3. 填写注册信息：
   - 用户名（唯一标识）
   - 邮箱（用于接收通知）
   - 密码（至少8位，包含字母和数字）
   - 显示名称（对外展示的名称）
4. 点击"注册"完成
5. 查收验证邮件并激活账号

### 登录与登出

**登录:**
1. 点击"登录"按钮
2. 输入用户名/邮箱和密码
3. 可选择"记住我"保持登录状态

**登出:**
- 点击右上角头像 → 选择"退出登录"

### 修改个人信息

1. 点击右上角头像 → "个人设置"
2. 可修改：
   - 显示名称
   - 邮箱（需重新验证）
   - 密码
   - 头像

### 忘记密码

1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接设置新密码

## 组织管理

### 什么是组织？

组织是 IPFlow 中的团队协作单位：
- 一个用户可加入多个组织
- 组织内的项目对所有成员可见
- 支持成员权限管理

### 创建组织

1. 点击左侧菜单"组织" → "创建组织"
2. 填写组织信息：
   - 组织名称（必填）
   - 组织标识（URL友好的唯一标识）
   - 描述（可选）
3. 选择订阅计划
4. 点击"创建"

### 切换组织

1. 点击顶部导航栏的组织名称
2. 在下拉菜单中选择要切换的组织
3. 系统会自动刷新显示该组织的数据

### 邀请成员

**管理员/经理可以邀请成员:**

1. 进入"组织设置" → "成员管理"
2. 点击"邀请成员"
3. 输入被邀请人的邮箱
4. 选择角色权限：
   - **管理员**: 完整组织管理权限
   - **经理**: 项目管理 + 成员邀请
   - **成员**: 创建/编辑项目
   - **查看者**: 只读访问
5. 点击"发送邀请"

被邀请人将收到邮件，点击链接即可加入。

### 管理成员权限

1. 在"成员管理"页面找到目标成员
2. 点击角色下拉框选择新角色
3. 权限变更立即生效

### 移除成员

1. 在"成员管理"页面找到目标成员
2. 点击"移除"按钮
3. 确认移除操作

> ⚠️ 注意：移除成员后，该成员创建的项目仍然保留在组织中。

## 项目管理

### 创建项目

1. 点击"新建项目"按钮
2. 选择申请类型：
   - 软件著作权（当前支持）
   - 专利（即将推出）
   - 商标（即将推出）
3. 填写基本信息：
   - 项目名称
   - 项目类型
   - 主体类型（个人/企业/机构）
   - 开发方式
   - 发表状态
4. 点击"创建"

### 项目列表

项目列表显示：
- 项目名称和版本
- 当前状态（草稿/进行中/已完成）
- 完成进度百分比
- 最后更新时间
- 快捷操作菜单

**筛选和搜索:**
- 按状态筛选
- 按类型筛选
- 关键词搜索

### 项目详情

点击项目卡片进入详情页，包含：
- **概览**: 项目基本信息和进度
- **软件信息**: 软件著作权的基本信息
- **源代码**: 代码鉴别材料处理
- **操作手册**: 操作说明书编辑
- **合规检查**: 材料完整性检查
- **导出**: 生成最终申请材料

### 删除项目

1. 在项目列表点击项目卡片的"更多"菜单
2. 选择"删除项目"
3. 确认删除操作

> ⚠️ 警告：删除后项目将进入回收站，30天后永久删除。

## 软件著作权申请

### 第一步：填写软件信息

**必填项:**
- **软件全称**: 与软件产品名称一致
- **软件版本号**: 如 V1.0、V2.5
- **完成日期**: 软件开发完成的日期
- **著作权人**: 法人或个人名称
- **著作权人类型**: 企业/个人/机构

**选填项:**
- **首次发表日期**: 如已发表需填写
- **城市**: 著作权人所在城市
- **软件描述**: 软件功能简介

**填写技巧:**
- 软件全称应包含品牌名 + 功能描述
- 版本号建议从 V1.0 开始
- 完成日期应在首次发表日期之前或相同

### 第二步：处理源代码

#### 上传代码

1. 进入"源代码"页面
2. 点击上传区域或拖拽文件
3. 支持的格式：
   - ZIP 压缩包（推荐）
   - RAR 压缩包
   - 7z 压缩包
4. 等待上传完成

#### 代码处理配置

系统默认配置（符合申请要求）：
- 前30页 + 后30页 = 60页
- 每页50行代码
- 自动添加行号
- 自动生成页眉（软件全称 + 版本号）

**可调整项:**
- 页数配置（需满足≥60页）
- 每页行数（需满足≥50行）
- 排除文件/目录

#### 预览和确认

1. 系统处理完成后显示代码预览
2. 检查：
   - 页眉是否正确
   - 行号是否连续
   - 代码是否完整
3. 确认无误后点击"确认使用"

> 💡 提示：代码鉴别材料要求是"前30页 + 后30页"，不是"前30页 + 后30页后的30页"。

### 第三步：编写操作手册

#### 选择模板

根据软件类型选择：
- **Web应用**: 包含首页、功能模块、系统管理
- **移动应用**: 包含启动页、主要功能、设置
- **算法软件**: 包含算法原理、输入输出、示例
- **脚本工具**: 包含安装、命令说明、示例
- **桌面软件**: 包含界面介绍、功能说明、操作步骤

#### 编辑内容

**必需章节:**
1. **封面**: 软件名称、版本号
2. **软件概述**: 功能简介、运行环境
3. **安装说明**: 安装步骤、环境配置
4. **使用说明**: 功能介绍、操作步骤
5. **截图**: 关键界面截图（≥5张）

**编辑技巧:**
- 使用工具栏快速格式化
- 点击"插入截图"上传图片
- 图片自动编号（图1、图2...）
- 实时查看字数和页数

**格式要求:**
- 不少于15页
- 30行/页
- 图片清晰，文字可辨认
- 图注完整（图X XXX界面）

### 第四步：合规检查

点击"开始检查"，系统将自动验证：

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 软件名称 | 不能为空 | ✓ |
| 软件版本 | 符合格式 | ✓ |
| 完成日期 | 合理日期 | ✓ |
| 著作权人 | 不能为空 | ✓ |
| 代码页数 | ≥60页 | ✓ |
| 代码行数 | ≥50行/页 | ✓ |
| 手册页数 | ≥15页 | ✓ |
| 手册截图 | ≥5张 | ✓ |
| 截图清晰度 | 可辨认 | ✓ |

**检查结果:**
- ✓ 通过: 该项符合要求
- ⚠ 警告: 建议优化
- ✗ 失败: 必须修正

### 第五步：导出申请材料

当所有检查项通过后：

1. 进入"导出"页面
2. 选择导出格式：
   - **标准ZIP包**: 包含所有申请材料
   - **单独PDF**: 分项导出
3. 点击"生成材料"
4. 等待生成完成
5. 点击"下载"

**导出文件清单:**
```
申请材料_软件名称_V1.0.zip
├── 01_申请表.pdf          # 在线填写的申请表
├── 02_源代码_鉴别材料.pdf  # 60页源代码
├── 03_操作说明书.pdf       # 操作手册
├── 04_身份证明/
│   ├── 企业_营业执照.pdf   # 企业申请需上传
│   └── 个人_身份证.pdf     # 个人申请需上传
└── 网报对照表.pdf          # 网上填报对照指南
```

### 提交申请

1. 登录中国版权保护中心官网
2. 在线填写申请表
3. 对照"网报对照表.pdf"逐项填写
4. 上传电子版材料
5. 邮寄纸质材料
6. 等待审核结果

> 📖 详细指南请参考：《软件著作权申请指南.pdf》

## 订阅与计费

订阅与计费是围绕材料生成、存储和协作限额提供的可选配套模块；具体可用性以部署配置为准。

### 计划类型

| 计划 | 价格 | 项目数 | 存储空间 | 成员数 |
|------|------|--------|----------|--------|
| 免费版 | ¥0 | 3个 | 100MB | 1人 |
| 专业版 | ¥99/月 | 无限 | 10GB | 10人 |
| 企业版 | ¥999/月 | 无限 | 100GB | 100人 |

### 查看使用统计

1. 点击"订阅管理"
2. 查看"使用情况"标签
3. 实时监控：
   - 项目使用量
   - 存储空间使用
   - 成员数量

### 升级计划

1. 进入"订阅管理"
2. 选择"订阅计划"标签
3. 选择目标计划
4. 选择计费周期（月付/年付）
5. 完成支付
6. 升级立即生效

### 取消订阅

1. 进入"订阅管理"
2. 点击"取消订阅"
3. 选择取消方式：
   - **到期取消**: 当前周期结束后停止，期间可继续使用
   - **立即取消**: 立即停止，按比例退款
4. 确认取消

### 发票管理

1. 进入"订阅管理" → "发票记录"
2. 查看历史发票
3. 点击"下载PDF"获取发票

## 常见问题

### 账号相关问题

**Q: 收不到验证邮件怎么办？**
- 检查垃圾邮件文件夹
- 确认邮箱地址输入正确
- 等待5-10分钟后重试
- 联系客服处理

**Q: 可以更换绑定的邮箱吗？**
- 可以，在"个人设置"中修改
- 修改后需要重新验证新邮箱

**Q: 一个邮箱可以注册多个账号吗？**
- 不可以，每个邮箱只能绑定一个账号

### 组织相关问题

**Q: 我可以创建多个组织吗？**
- 可以，没有数量限制
- 每个组织独立计费

**Q: 组织创建者可以退出组织吗？**
- 需要先转让所有权或删除组织
- 不能直接退出

**Q: 如何转让组织所有权？**
- 在"组织设置" → "成员管理"
- 找到目标成员，将其设为管理员
- 然后退出组织

### 项目相关问题

**Q: 项目数量达到上限怎么办？**
- 删除不需要的旧项目
- 或升级订阅计划

**Q: 可以复制项目吗？**
- 暂不支持直接复制
- 可以导出材料后新建项目导入

**Q: 上传的代码文件安全吗？**
- 数据采用加密存储
- 只有组织成员可以访问
- 支持随时删除

### 代码处理相关问题

**Q: 支持哪些编程语言？**
- 支持所有文本格式的源代码
- 自动识别语言类型
- 代码高亮显示

**Q: 代码文件大小限制是多少？**
- 单个文件最大 500MB
- 压缩包最大 1GB
- 如需更大请联系客服

**Q: 代码抽取结果不符合预期？**
- 检查"前N页+后N页"配置
- 查看排除的文件列表
- 确认压缩包结构

### 操作手册相关问题

**Q: 手册页数不够15页怎么办？**
- 详细描述软件功能
- 增加操作步骤说明
- 添加更多界面截图
- 补充安装配置说明

**Q: 截图有什么要求？**
- 分辨率建议 1920x1080 以上
- 图片清晰，文字可辨认
- 标注关键操作区域
- 按操作流程排序

**Q: 可以导入Word文档吗？**
- 暂不支持直接导入
- 可以复制粘贴内容到编辑器

### 导出相关问题

**Q: 导出失败怎么办？**
- 检查所有检查项是否通过
- 确认网络连接正常
- 尝试重新生成
- 联系客服协助

**Q: 生成的PDF可以修改吗？**
- 不建议直接修改
- 如需修改请在系统中编辑后重新导出
- 确保申请材料一致性

### 计费相关问题

**Q: 年付和月付有什么区别？**
- 年付享受8折优惠
- 功能完全相同
- 可随时更改周期

**Q: 可以免费试用吗？**
- 免费版可永久使用
- 付费计划不提供试用
- 可随时取消退款

**Q: 退款政策是什么？**
- 7天内无理由退款
- 按比例退还剩余时长
- 发票需退回
