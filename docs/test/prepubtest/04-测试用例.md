# IPFlow 上线前测试 — 测试用例（Test Cases）

> 由用户故事（US-1~US-6）派生。每条含**数据预期**。执行时照用例操作→观察→验数据，覆盖完打勾。
> 优先级：🔴P0 核心·必过 🟠P1 🟡P2。类型：功能/数据/异常/边界。
> 用例 ID 由 `log-session.sh testcase` 生成（TC-1 起）。
