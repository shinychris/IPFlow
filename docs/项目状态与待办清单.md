# 知识产权助手 - 项目状态与待办清单

> 更新日期：2026年2月10日

---

## 一、已完成功能

### 1. 用户认证系统
- 用户注册（用户名、昵称、密码）
- 用户登录/退出
- 会话管理（PostgreSQL 持久化存储）
- 密码加密存储（bcrypt）
- 路由保护（未登录自动跳转登录页）
- 登录/注册页面左右分栏布局（左侧图片+标题，右侧表单）

### 2. 仪表盘（工作台）
- 三种业务类型入口卡片（软著、专利、商标）
- 项目数量统计（按类型、按状态）
- 最近项目列表展示
- 新建项目快捷入口

### 3. 项目管理
- 项目列表（全部项目 + 按类型筛选页面）
- 搜索与筛选（按名称搜索、按状态/类型筛选）
- 创建项目、编辑项目、删除项目、复制项目
- 项目状态管理（草稿、进行中、已完成、已导出）

### 4. 软著申请向导（5步）
- 第1步：创建项目（主体类型、开发方式、发布状态）
- 第2步：软件信息（全称、简称、版本号、开发语言、环境、功能描述等）
- 第3步：代码材料（上传 ZIP 源代码，自动提取前30页+后30页共60页，每页50行）
- 第4步：文档材料（操作说明书编辑器，支持5种模板类型）
- 第5步：导出交付（合规检查清单 + ZIP包导出）

### 5. 专利申请向导（5步）
- 第1步：创建项目
- 第2步：专利信息（专利类型、发明名称、摘要、申请人信息等）
- 第3步：权利要求（权利要求书编写）
- 第4步：说明书（技术领域、背景技术、技术问题/方案/效果、附图说明）
- 第5步：导出交付

### 6. 商标申请向导（4步）
- 第1步：创建项目
- 第2步：商标信息（商标类型、名称、申请人信息、联系方式等）
- 第3步：商品服务（尼斯分类选择，支持45类商品服务）
- 第4步：导出交付

### 7. 辅助页面
- 申请规范页面（三种业务类型的合规规则说明）
- 帮助中心页面（常见问题、流程指南）
- 侧边栏导航（显示当前用户、退出按钮）

### 8. 文件管理
- 源代码文件上传与解析（ZIP包）
- 证明材料上传与下载（对象存储）
- 文件大小限制（100MB）

## 二、存在的问题

### 严重问题

| 编号 | 问题描述 | 影响范围 |
|------|----------|----------|
| S-1 | **项目未绑定用户**：projects 表没有 userId 字段，所有用户共享所有项目，无法做数据隔离 | 数据安全 |
| S-2 | **导出仅支持软著**：export-generator.ts 只生成软著申请包（ZIP包含源代码、说明书、软件信息等），专利和商标的导出会生成不完整的包 | 专利/商标 |

### 一般问题

| 编号 | 问题描述 | 影响范围 |
|------|----------|----------|
| M-1 | **导出格式为纯文本**：导出的 ZIP 内所有文件为 .txt 格式，不是正式的 PDF 格式（无页眉页脚、无签章位） | 文档质量 |
| M-2 | **"生成打印指南"和"查看网报对照表"按钮无功能**：导出步骤中的两个辅助按钮点击后没有任何反应 | 用户体验 |
| M-3 | **合规检查仅为前端状态**：合规检查结果主要在前端维护，服务端的合规检查逻辑简单，未真正校验各项数据是否满足要求 | 数据准确性 |
| M-4 | **操作说明书编辑器功能有限**：文档编辑器为基础文本编辑，缺少富文本格式、截图插入、自动排版等功能 | 文档质量 |
| M-5 | **证明材料未包含在导出包中**：导出的 ZIP 包中只有证明材料清单文本，未包含实际上传的文件 | 完整性 |

### 轻微问题

| 编号 | 问题描述 | 影响范围 |
|------|----------|----------|
| L-1 | 专利向导中发明人字段为逗号分隔文本输入，不够直观 | 用户体验 |
| L-2 | 商标图样上传字段（imageObjectPath）已定义但向导中未集成上传功能 | 商标功能 |
| L-3 | 页面缺少加载错误提示（API 失败时部分页面无反馈） | 用户体验 |
| L-4 | 没有用户管理后台（管理员角色已定义但无对应管理页面） | 管理功能 |

## 四、技术架构概要

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + TypeScript |
| 路由 | Wouter |
| UI组件 | shadcn/ui (Radix UI) |
| 样式 | Tailwind CSS |
| 状态管理 | TanStack React Query |
| 后端 | Node.js + Express |
| 数据库 | PostgreSQL (Drizzle ORM) |
| 文件存储 | Replit Object Storage |
| 认证 | express-session + bcryptjs |
| 构建工具 | Vite (前端) + esbuild (后端) |
