## 🚀 快速开始

### 我是用户

1. 阅读 [用户手册](user-guide.md) 了解基本使用
2. 查看 [CHANGELOG](../CHANGELOG.md) 了解最新功能

### 我是开发者

1. 阅读 [架构设计](architecture.md) 了解系统架构
2. 按照 [开发指南](development.md) 搭建环境
3. 参考 [API 端点速查](API_ENDPOINTS.md) 查看所有接口
4. 查看 [API 文档](api.md) 了解详细用法

### 我是运维

1. 阅读 [部署指南](deployment.md) 进行部署
2. 参考 [数据库文档](database.md) 进行数据库管理

### 我是产品经理

1. 阅读 [产品说明文档](../IPFlow-产品说明文档.md) 了解产品定位
2. 查看 [架构设计文档](../docs/IPFlow-v2-架构设计文档.md) 了解技术实现
3. 参考 [AI 辅助功能设计](AI辅助功能设计.md) 规划未来功能

## 📖 功能模块

### 已完成功能 ✅

- **软件著作权** - 5步向导，代码处理，说明书编辑，合规检查，导出
- **专利申请** - 发明/实用新型/外观设计，权利要求书，说明书，附图
- **商标注册** - 6种商标类型，尼斯分类45类，图样上传
- **组织管理** - 多租户，成员管理，邀请机制，角色权限
- **订阅系统** - 5种计划，限额控制，使用统计
- **管理后台** - 仪表盘，用户/组织/计划管理，审计日志

### 计划功能 📅

- PDF 导出功能完善
- AI 辅助撰写
- 支付提供商集成
- 邮件服务
- 移动端 App
