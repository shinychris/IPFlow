# 中国商标申请材料编写助手

协助用户编写符合中国国家知识产权局商标局要求的中国商标注册申请文件。

## 商标类型判断

首先判断用户需要申请的商标类型：

| 商标类型 | 适用对象 | 特点 |
|----------|----------|------|
| **普通商标** | 自然人、法人或其他组织 | 最常见的商标类型，用于区分商品/服务来源 |
| **集体商标** | 团体、协会或其他组织 | 供组织成员使用，表明成员资格 |
| **证明商标** | 具有监督能力的组织 | 证明商品/服务的特定品质、原料、制造方法等 |
| **地理标志** | 当地不以盈利为目的的组织 | 标示商品来源于某地区，具有特定品质、声誉 |

---

## 使用模式

### 模式一：材料整理模式

用户已提供商标设计、经营范围等基础材料，需要整理成规范的申请文件。

**工作流程：**
```
确认商标信息 → 尼斯分类选择 → 商品/服务项目确定 → 申请书填写 → 材料清单核对
```

### 模式二：自由生成模式

用户仅有商标名称或初步想法，需要指导完成整个申请流程。

**信息收集：**
- 商标名称/图样
- 申请人信息（个人/企业）
- 经营范围/服务范围
- 使用或意图使用的商品/服务

## 尼斯分类与商品/服务项目

### 尼斯分类概览

中国采用《商标注册用商品和服务国际分类》（尼斯分类），共分为**45个类别**：

| 类别范围 | 内容 |
|----------|------|
| **第1-34类** | 商品（化工、医药、食品、服装、机械、电子产品等） |
| **第35-45类** | 服务（广告、金融、运输、教育、法律、餐饮等） |

**现行版本：** 尼斯分类第十二版2025文本（2025年1月1日起适用）

### 核心类别速查表

| 行业 | 核心类别 | 相关类别 |
|------|----------|----------|
| 服装鞋帽 | 第25类 | 第18类（箱包）、第35类（销售服务） |
| 食品饮料 | 第29类（食品）、第30类（方便食品）、第31类（生鲜） | 第32类（饮料）、第33类（酒）、第43类（餐饮服务） |
| 化妆品 | 第3类 | 第35类（销售服务） |
| 医药保健 | 第5类（药品）、第10类（医疗器械） | 第35类（药店零售）、第44类（医疗服务） |
| 电子产品 | 第9类 | 第11类（家用电器）、第37类（维修服务） |
| 软件开发 | 第9类（软件）、第42类（技术服务） | 第35类（广告）、第38类（电信） |
| 教育培训 | 第41类 | 第35类（推广）、第42类（在线教育平台） |
| 餐饮服务 | 第43类 | 第29类（食品）、第30类（调料）、第35类（连锁管理） |
| 金融服务 | 第36类 | 第35类（金融咨询）、第42类（金融科技平台） |
| 物流运输 | 第39类 | 第35类（物流管理）、第42类（物流平台） |

### 商品/服务项目选择原则

1. **选择标准名称**
   - 优先使用《类似商品和服务区分表》中已列出的标准名称
   - 参考网址：https://sbj.cnipa.gov.cn/sbj/sbsq/sphfwfl/

2. **一个类别一份申请**
   - 一标多类申请：一份申请可包含多个类别（官费按类别收取）
   - 每类可选择10个商品/服务项目，超出需额外缴费

3. **覆盖当前使用和预期使用**
   - 包含当前实际使用的项目
   - 包含未来1-3年可能拓展的业务范围

4. **避免过于宽泛**
   - 不可申报类别标题（如"化妆品"需具体到"护肤用化妆剂"）
   - 不可申报含混不清的名称（如"电子商务服务"需具体化）

## 申请文件要求

### 必备文件清单

| 序号 | 文件名称 | 说明 |
|------|----------|------|
| 1 | **商标注册申请书** | 在线填写或下载填写 |
| 2 | **商标图样** | JPG格式，清晰、不大于2M |
| 3 | **申请人身份证明** | 营业执照（企业）或身份证（个人） |
| 4 | **代理委托书** | 委托代理机构办理时需提供 |

### 申请人资格要求

**企业申请：**
- 提供营业执照副本复印件（加盖公章）
- 申请名义与营业执照一致

**个人申请：**
- 提供个体工商户营业执照复印件（加盖公章）
- 提供个人身份证复印件（签字）
- 申请名义与执照经营者一致

**特殊主体：**
- 集体商标：团体、协会或其他组织
- 证明商标：具有监督能力的组织
- 地理标志：当地不以盈利为目的的组织

### 商标图样要求

| 项目 | 要求 |
|------|------|
| **格式** | JPG格式 |
| **尺寸** | 400×400至1500×1500像素 |
| **大小** | 不大于2MB |
| **清晰度** | 清晰完整，不模糊 |
| **背景** | 纯色背景（黑白商标除外） |
| **颜色** | 彩色或黑白，需与申请时声明一致 |

**特殊商标类型：**

| 类型 | 特殊要求 |
|------|----------|
| **三维标志** | 提交能够确定三维形状的图样，至少3个视图 |
| **颜色组合** | 提交着色图样，并说明颜色组合方式 |
| **声音商标** | 提交声音样本（WAV或MP3，不超过5MB）和乐谱 |
| **肖像商标** | 提交肖像权人授权书并经公证 |

## 申请书填写要点

### 基本信息

| 项目 | 填写说明 |
|------|----------|
| **商标名称** | 中文、英文、图形或其组合 |
| **商标说明** | 对商标的说明（如特殊字体、放弃专用权部分等） |
| **类别** | 尼斯分类类别号（1-45） |
| **商品/服务** | 具体项目名称（每类10项以内） |

### 商标说明填写示例

```
1. 放弃专用权说明：
   "本商标中'XX'文字放弃专用权"

2. 特殊字体说明：
   "该商标文字为艺术字体，具体样式见图样"

3. 三维标志说明：
   "该商标为三维标志，具体形状见三维视图"

4. 颜色组合说明：
   "该商标为颜色组合商标，由红色（Pantone 185C）和
    黄色（Pantone 123C）组成"
```

## 申请流程

```
1. 商标查询 → 2. 准备材料 → 3. 提交申请 → 4. 形式审查 → 5. 实质审查
                                                   ↓              ↓
                                              补正（如需）   驳回/部分驳回
                                                   ↓              ↓
                                              受理通知书   复审（可选）
                                                                  ↓
6. 初步审定公告（3个月）→ 7. 注册公告 → 8. 颁发注册证
              ↓
        异议（如有）
              ↓
        异议答辩/裁定
```

### 时间节点

| 阶段 | 时间 |
|------|------|
| 形式审查 | 1-2个月 |
| 受理通知书 | 形式审查通过后 |
| 实质审查 | 4-6个月 |
| 初步审定公告 | 3个月异议期 |
| 注册公告 | 异议期满无异议 |
| 总周期 | 约9-12个月（无驳回/异议） |

### 费用参考

| 项目 | 官费（电子申请） | 官费（纸质申请） |
|------|------------------|------------------|

## 商标查询建议

### 查询目的

1. **降低驳回风险**：查询是否有相同/近似在先商标
2. **调整申请策略**：根据查询结果调整商标设计或类别
3. **评估注册成功率**：预先评估注册可能性

### 查询渠道

| 渠道 | 网址 | 特点 |
|------|------|------|
| 中国商标网 | https://sbj.cnipa.gov.cn | 官方，免费 |
| 商标局查询大厅 | - | 可出具查询证明 |
| 第三方平台 | 权大师、知擎等 | 功能丰富，部分收费 |

### 查询要点

1. **相同查询**：检索完全相同的商标
2. **近似查询**：检索音、形、义近似的商标
3. **类别限制**：在相同/类似商品/服务上检索
4. **状态检查**：关注在先商标的法律状态

## 常见驳回原因及应对

### 绝对理由驳回

| 驳回原因 | 说明 | 应对策略 |
|----------|------|----------|
| **缺乏显著性** | 仅直接表示商品质量、原料、功能等 | 修改商标设计增加显著性 |
| **不良影响** | 带有欺骗性、不良影响或违法内容 | 更换商标 |
| **与国旗国徽等相同/近似** | 同中华人民共和国的国家名称、国旗等相同或近似 | 更换商标 |
| **与官方标志相同/近似** | 同政府间国际组织的名称、旗帜等相同或近似 | 更换商标或取得授权 |

### 相对理由驳回

| 驳回原因 | 说明 | 应对策略 |
|----------|------|----------|
| **与在先商标相同/近似** | 与他人在同一种或类似商品上的在先商标相同/近似 | 修改商标、更换类别、转让/撤销在先商标 |
| **抢注他人在先使用商标** | 抢注他人在先使用并有一定影响的商标 | 提供在先使用证据或协商转让 |

### 驳回复审

- **期限**：收到驳回通知15日内
- **费用**：750元
- **成功率**：视具体情况，约30-50%

## 商标使用与维护

### 商标使用规范

1. **按注册图样使用**：不得自行改变注册商标的文字、图形等
2. **按核定类别使用**：不得超范围使用
3. **标注注册标记**：可使用®标记（注册后）
4. **保留使用证据**：合同、发票、广告等

### 商标续展

- **有效期**：10年
- **续展期**：期满前12个月内
- **宽展期**：期满后6个月内（加收延迟费）
- **续展费用**：450元（电子申请）

### 商标变更

以下情况需办理变更：
- 申请人名义变更
- 申请人地址变更
- 代理人变更

### 商标转让

- **转让方式**：合同转让、继承转让、因企业合并/分立转让
- **转让费用**：450元（电子申请）
- **转让周期**：约4-6个月

## 重要提醒

1. **在先使用**：商标注册遵循申请在先原则，尽早申请
2. **使用义务**：注册商标无正当理由连续3年不使用可被撤销
3. **监测维护**：定期监测是否有近似商标申请，及时异议
4. **续展及时**：到期前及时续展，避免权利丧失
5. **规范使用**：按注册图样和核定类别规范使用
