# 中国商标申请详细要求

## 一、尼斯分类体系

### 1.1 尼斯分类简介

《商标注册用商品和服务国际分类》（尼斯分类）将商品和服务分为**45个类别**：

- **第1-34类**：商品
- **第35-45类**：服务

尼斯分类每年修订一次，中国商标局以尼斯分类为基础，制定《类似商品和服务区分表》。

**现行版本：** 尼斯分类第十二版2025文本（2025年1月1日起适用）

### 1.4 商品/服务项目选择规范

#### 选择标准名称

**优先使用《类似商品和服务区分表》中的标准名称**

查询网址：https://sbj.cnipa.gov.cn/sbj/sbsq/sphfwfl/

#### 申报规范示例

| 不规范 | 规范 |
|--------|------|
| 化妆品（太宽泛） | 护肤用化妆剂、口红、香水等具体项目 |
| 电子产品（太宽泛） | 智能手机、计算机外围设备、耳机等 |
| 电子商务服务（太宽泛） | 广告、为零售目的在通讯媒体上展示商品等 |
| 食品（跨多类） | 分具体类别：肉（29类）、咖啡（30类）、水果（31类） |
| 钢及其合金（包含多个商品） | 分别申报：钢；钢合金 |

#### 非规范商品/服务申报

如《区分表》中无适合项目，可申报非规范项目，但需满足：
1. 符合尼斯分类原则
2. 名称具体明确，可与其他类别区分
3. 不得使用含混不清、过于宽泛的名称
4. 不得使用违反法律法规或有不良影响的词语

## 二、申请文件详细要求

### 2.1 商标注册申请书

**填写要求：**
- 使用商标局统一格式的申请书
- 内容真实、准确、完整
- 字迹清晰，不得涂改
- 申请人签字或盖章

**主要内容：**

| 项目 | 填写说明 |
|------|----------|
| 申请人名称 | 与身份证明文件一致 |
| 申请人地址 | 详细通讯地址 |
| 联系人 | 便于联系的人员 |
| 联系电话 | 有效联系电话 |
| 商标名称 | 申请注册的商标文字/图形名称 |
| 商标说明 | 对商标的说明（如放弃专用权、颜色组合等） |
| 类别 | 尼斯分类类别号 |
| 商品/服务项目 | 具体项目名称 |
| 是否集体/证明商标 | 如是，需选择并提供相应证明文件 |
| 优先权声明 | 如要求优先权，需填写在先申请信息 |

### 2.2 商标图样要求

#### 一般要求

| 项目 | 具体要求 |
|------|----------|
| 文件格式 | JPG |
| 像素尺寸 | 400×400至1500×1500像素 |
| 文件大小 | 不大于2MB |
| 分辨率 | 清晰可辨，不小于72dpi |
| 颜色模式 | RGB或灰度 |
| 背景 | 纯色（推荐白色），与商标形成明显对比 |

#### 彩色商标与黑白商标

| 类型 | 说明 | 使用范围 |
|------|------|----------|
| **彩色商标** | 申请时指定颜色，按指定颜色保护 | 仅保护指定颜色组合 |
| **黑白商标** | 不指定颜色，按黑白保护 | 实际可使用任意颜色 |

**建议：** 除非颜色是商标显著特征，否则建议申请黑白商标，保护范围更广。

#### 特殊商标图样要求

**三维标志商标：**
- 提交三维视图（至少3个视图：主视图、俯视图、左视图）
- 或提交立体图
- 应在申请书中声明"以三维标志申请商标注册"

**颜色组合商标：**
- 提交着色图样
- 说明颜色组合方式
- 申报指定色标（如Pantone色号）

**声音商标：**
- 提交声音样本（WAV或MP3格式，不超过5MB）
- 提交五线谱或简谱
- 文字说明声音并描述听觉效果

### 2.3 身份证明文件

#### 企业申请

| 文件 | 要求 |
|------|------|
| 营业执照副本 | 复印件，加盖公章 |
| 有效期 | 申请时营业执照在有效期内 |
| 一致性 | 申请人名称与营业执照完全一致 |

#### 个人申请

| 文件 | 要求 |
|------|------|
| 个体工商户营业执照 | 复印件，加盖公章 |
| 个人身份证 | 复印件，签字 |
| 一致性 | 申请人名义与执照经营者一致 |

#### 外国申请人

| 文件 | 要求 |
|------|------|
| 主体资格证明 | 经所在国公证机关公证并经中国驻该国使领馆认证 |
| 中文翻译 | 提交中文翻译件 |
| 代理委托 | 必须委托中国商标代理机构办理 |

### 2.4 代理委托书

**委托代理机构办理时需提供：**

| 项目 | 要求 |
|------|------|
| 委托人 | 申请人签字或盖章 |
| 受托人 | 代理机构名称、代理人姓名 |
| 委托事项 | 商标注册申请及相关事宜 |
| 权限 | 明确代理权限范围 |

## 三、特殊类型商标要求

### 3.1 集体商标

**申请人资格：**
- 必须是团体、协会或其他组织
- 不能是单一企业或个体经营者

**使用管理规则：**
- 使用集体商标的宗旨
- 使用该商标的集体成员
- 使用条件、手续
- 成员权利义务
- 使用管理费用
- 品质保证措施
- 监督检查机制

### 3.2 证明商标

**申请人资格：**
- 必须是对商品/服务具有监督能力的组织
- 不得从事相关商品/服务的经营

**使用管理规则：**
- 证明商标证明的内容
- 使用条件、手续
- 检测和监督制度
- 违规处理措施

**示例：** 
- 绿色食品标志
- 纯羊毛标志
- 真皮标志

### 3.3 地理标志

**申请人资格：**
- 当地不以盈利为目的的团体、协会或其他组织
- 一般是社会团体法人、事业单位法人
- 业务范围与地理标志产品相关

**申请材料：**
- 主体资格证明文件
- 地理标志所标示地区县级以上人民政府的授权文件
- 使用管理规则
- 地理标志产品的特定质量、信誉或其他特征说明
- 产品的生产地域范围界定文件

**示例：**
- 阳澄湖大闸蟹
- 西湖龙井
- 金华火腿

### 3.4 肖像商标

**特殊要求：**
- 提交肖像权人授权书
- 授权书需经公证
- 自然人本人申请使用自己肖像的，无需授权

## 四、电子申请要求

### 4.1 申请途径

| 途径 | 说明 | 网址/地址 |
|------|------|-----------|
| 商标网上服务系统 | 在线申请（推荐） | https://sbj.cnipa.gov.cn/wssq/ |
| 商标业务受理窗口 | 各地商标受理窗口 | 各地市场监管部门 |
| 商标审查协作中心 | 京、沪、广、渝、津、郑、济 | 各中心地址 |
| 商标注册大厅 | 国家知识产权局商标局 | 北京市西城区茶马南街1号 |

### 4.2 电子申请优势

- 官费优惠（270元/类 vs 300元/类）
- 申请进度可在线查询
- 电子发文，快捷便利
- 不受办公时间限制

### 4.3 数字证书申请

使用商标网上服务系统需先申请数字证书：
1. 在线填写申请信息
2. 提交身份证明文件
3. 审核通过后下载数字证书

## 五、优先权要求

### 5.1 外国优先权

**条件：**
- 在先申请为首次申请
- 在6个月内就相同商标向中国提出申请
- 商标和商品/服务相同

**手续：**
- 申请时声明优先权
- 提交优先权证明文件（经公证认证）
- 3个月内提交中文翻译件

### 5.2 展会优先权

**条件：**
- 在中国政府主办或承认的国际展览会展出
- 6个月内提出申请
- 提交展会证明文件

## 六、国际注册（马德里体系）

### 6.1 马德里国际注册简介

通过世界知识产权组织（WIPO）国际局，一件申请可指定多个国家。

### 6.2 申请条件

- 在中国有真实有效的工商营业场所
- 或在中国有住所
- 或具有中国国籍

### 6.3 申请方式

**国内基础申请/注册：**
- 先在国内申请或获得注册
- 以国内申请/注册为基础提交国际申请

**申请途径：**
- 通过商标局提交（线上或线下）
- 官费+国际规费

### 6.4 优点

- 一份申请，多国保护
- 费用相对较低
- 统一管理、续展

### 6.5 缺点

- 中心打击原则：基础申请/注册失效，国际注册也失效
- 审查周期较长（12-18个月）
- 部分国家可能有特殊要求
