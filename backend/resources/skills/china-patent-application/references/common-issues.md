# 中国专利申请常见问题与审查意见答复

## 一、常见审查意见类型

### 1.1 新颖性审查意见

**常见表述：**
- "权利要求X不具备专利法第22条第2款规定的新颖性"
- "对比文件1公开了...，与权利要求X的技术方案相同"

**答复策略：**
1. 核实对比文件是否确实公开了本申请的全部技术特征
2. 寻找区别特征（即使是很小的区别）
3. 如确有区别，修改权利要求突出该区别，并说明该区别带来的效果
4. 如确实被公开，考虑增加未被公开的技术特征

**示例答复：**
```
申请人认为，权利要求1相对于对比文件1具有新颖性。

对比文件1公开了[描述]，但是未公开本申请权利要求1中的[区别特征]。
因此，权利要求1的技术方案与对比文件1公开的内容不同，具备新颖性。
```

### 1.2 创造性审查意见

**常见表述：**
- "权利要求X不具备专利法第22条第3款规定的创造性"
- "对比文件1给出了技术启示"
- "属于本领域的常规技术手段"

**答复策略：**
1. 强调区别特征的技术效果
2. 说明对比文件之间没有结合的技术启示
3. 提供实验数据证明有益效果
4. 证明区别特征不是本领域的公知常识

**示例答复：**
```
申请人认为，权利要求1相对于对比文件1和对比文件2的组合具有创造性。

首先，对比文件1涉及[领域]，而本申请解决的是[另一技术问题]，两者技术
领域不同，技术问题也不同。

其次，虽然对比文件2公开了[某特征]，但是对比文件2中没有给出将该特征
应用于对比文件1的技术启示。本领域技术人员没有动机将对比文件2的技术
方案结合到对比文件1中。

此外，本申请通过[区别特征]实现了[意想不到的技术效果]，这种效果在现
有技术中并未记载，也不是本领域技术人员可以预期的。

因此，权利要求1具有突出的实质性特点和显著的进步，具备创造性。
```

### 1.3 公开不充分审查意见

**常见表述：**
- "说明书未对发明作出清楚、完整的说明"
- "本领域技术人员无法实施"
- "缺少必要的实施例"

**答复策略：**
1. 补充详细的实施例
2. 补充实验数据或操作参数
3. 补充附图说明
4. 解释清楚工作原理

### 1.4 权利要求不清楚审查意见

**常见表述：**
- "权利要求X不清楚，不符合专利法第26条第4款的规定"
- "术语[XXX]含义不清楚"
- "保护范围不清楚"

**答复策略：**
1. 修改不清楚的术语，使用规范的技术用语
2. 删除模糊表述（如"等"、"各种"）
3. 明确数值范围
4. 确保引用关系正确

## 二、专利撰写常见问题

### 2.1 说明书常见问题

| 问题 | 示例 | 修改建议 |
|------|------|----------|
| 商业宣传用语 | "本发明具有划时代的意义" | 删除，改为"本发明具有[具体技术效果]" |
| 引用权利要求 | "如权利要求1所述的..." | 改为"所述的[具体特征]" |
| 缺少实施例 | 只有一个实施例，无变化例 | 补充2-3个不同实施方式 |
| 参数范围过宽 | "温度范围为0-1000℃" | 给出优选范围并说明理由 |
| 缺少实验数据 | 声称有效果但无数据 | 补充对比实验或效果数据 |

### 2.2 权利要求书常见问题

| 问题 | 示例 | 修改建议 |
|------|------|----------|
| 非必要技术特征 | 将非核心特征写入独立权利要求 | 移至从属权利要求 |
| 功能性限定 | "用于...的装置" | 改为结构特征描述 |
| 多项引用多项 | "根据权利要求1-3任一项..."作为另一多项引用的基础 | 改为择一引用 |
| 缺少引用基础 | 引用权利要求1，但权利要求1无该技术特征 | 核实并修正引用关系 |
| 保护范围过窄 | 写入过多限制性特征 | 提炼必要技术特征 |
| 保护范围过宽 | 上位概念无法支持 | 补充下位概念实施例 |

### 2.3 附图常见问题

| 问题 | 修改建议 |
|------|----------|
| 附图标记不一致 | 统一全文附图标记 |
| 文字部分有附图标记但附图中无 | 在附图中添加或删除文字中的标记 |
| 附图中有文字说明 | 删除附图中的文字，改为标记 |
| 缺少必要附图 | 补充结构示意图或流程图 |
| 附图不清晰 | 重新绘制，确保线条清晰 |

## 三、不同类型专利的注意事项

### 3.1 发明专利

**注意事项：**
- 必须具备突出的实质性特点和显著的进步
- 说明书需要充分公开，使本领域技术人员能够实现
- 权利要求以说明书为依据
- 18个月后公开，可请求提前公开
- 需在申请日起3年内提出实质审查请求

**加速审查途径：**
| 途径 | 周期 | 条件 |
|------|------|------|
| 优先审查 | 1年内结案 | 涉及国家重点发展产业等 |
| 快速预审 | 3-6个月 | 通过保护中心预审 |
| PPH | 加快 | 对应申请已在其他局获权 |

### 3.2 实用新型专利

**注意事项：**
- 必须有附图
- 保护期限10年
- 初步审查制，不进行实质审查
- 创造性要求低于发明
- 仅保护产品，不保护方法

**常见缺陷：**
- 说明书附图不能清晰显示要求保护的产品的形状、构造
- 权利要求中包含方法特征
- 权利要求中包含材料特征（未对结构产生影响）

### 3.3 外观设计专利

**注意事项：**
- 提交六面视图和立体图（必要时）
- 图片或照片应清晰显示产品外观设计
- 简要说明应准确描述设计要点
- 保护期限15年

**常见缺陷：**
- 图片中包含多余内容（如背景、阴影）
- 视图比例不一致
- 简要说明与设计要点不符
- 省略视图未说明理由

## 四、专利申请流程注意事项

### 4.1 申请日确定

- 直接向专利局递交的，以收到日为申请日
- 邮寄的，以寄出邮戳日为申请日
- 电子申请的，以系统接收日为申请日

### 4.2 优先权

**要求优先权的条件：**
- 在先申请为首次申请
- 在12个月（发明/实用新型）或6个月（外观设计）内提出
- 请求书中声明并写明在先申请信息
- 提交优先权证明文件（必要时）

### 4.3 分案申请

**分案时机：**
- 原申请授权前均可提出
- 原申请视为撤回或驳回后2个月内可提出

**分案要求：**
- 不得超出原申请记载的范围
- 保留原申请日，有优先权的保留优先权日
- 类别应与原申请一致（发明/实用新型可互转）

### 4.4 费用缴纳

**主要费用及期限：**

| 费用名称 | 缴纳期限 | 金额（参考） |
|----------|----------|--------------|
| 申请费 | 申请日起2个月内或收到受理通知书15日内 | 发明900元，实用新型/外观500元 |
| 实质审查费 | 申请日起3年内 | 发明2500元 |
| 年费 | 授权当年起每年缴纳 | 逐年递增 |
| 滞纳金 | 年费期满6个月内 | 每月5% |

**费用减缓：**
- 个人：最高减缓85%
- 单位：最高减缓70%
- 需提交费用减缓请求书及相关证明

## 五、专利检索建议

### 5.1 检索时机

1. **申请前**：评估新颖性和创造性
2. **撰写时**：了解现有技术，确定区别特征
3. **答复审查意见时**：寻找对比文件缺陷或补充对比文件

### 5.2 常用检索数据库

| 数据库 | 网址 | 特点 |
|--------|------|------|
| 中国专利公布公告 | http://epub.cnipa.gov.cn | 官方，权威 |
| 专利检索及分析系统 | http://pss-system.cponline.cnipa.gov.cn | 功能全面 |
| 佰腾网 | https://www.baiten.cn | 免费，易用 |
|  incoPat | https://www.incopat.com | 商业数据库，功能强大 |
| 专利之星 | http://www.patentstar.cn | 免费检索 |

### 5.3 检索关键词策略

1. **分类号检索**：IPC/CPC分类号
2. **关键词检索**：同义词、近义词、上下位概念
3. **申请人/发明人检索**：追踪竞争对手
4. **引证检索**：向前/向后引证

## 六、专利维持与运用

### 6.1 年费缴纳

- 授权当年开始缴纳年费
- 年费逐年递增
- 期满未缴纳可补缴并缴纳滞纳金（6个月宽限期）

### 6.2 专利权评价报告

- 实用新型/外观设计授权后可请求
- 用于侵权纠纷中的权利稳定性证明
- 由专利局出具

### 6.3 专利转让与许可

**转让：**
- 需向专利局登记并公告
- 自登记之日起生效

**许可：**
- 独占许可、排他许可、普通许可
- 建议在国家知识产权局备案

### 6.4 专利维权

**侵权判定：**
- 全面覆盖原则
- 等同原则
- 禁止反悔原则

**维权途径：**
- 行政途径：向地方知识产权局投诉
- 司法途径：向法院起诉
- 协商调解
