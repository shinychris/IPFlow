## 说明书

### 发明名称

一种基于自适应PID算法的智能温控系统

### 技术领域

本发明涉及智能家居技术领域，具体涉及一种基于自适应PID算法的智能温控系统。

### 背景技术

温度控制系统广泛应用于家用电器、工业设备、农业大棚等领域。传统的温度控制系统多采用固定参数的PID（比例-积分-微分）控制算法，虽然结构简单、稳定性好，但在实际应用中存在以下问题：

1. **参数整定困难**：传统PID控制器的比例系数Kp、积分系数Ki、微分系数Kd需要通过经验或试凑法确定，对于非线性、时变系统难以获得最优参数。

2. **适应性差**：当系统工况发生变化（如环境温度大幅波动、负载变化）时，固定参数的PID控制器难以保持良好的控制性能，容易出现超调大、响应慢、振荡等问题。

3. **能耗较高**：为保持控制精度，传统系统往往采用频繁启停或持续高功率运行的方式，造成能源浪费。

针对上述问题，中国专利CN201910123456.7公开了一种模糊PID温度控制方法，通过模糊推理在线调整PID参数，提高了系统的适应性。但是该方法存在计算复杂度高、对模糊规则设计依赖性强的问题，且未考虑系统的预测控制。

因此，亟需一种能够自动适应工况变化、控制精度高、能耗低的智能温控系统。

### 发明内容

本发明旨在至少解决现有技术中存在的技术问题之一。为此，本发明提出一种基于自适应PID算法的智能温控系统，能够根据环境变化自动调整控制参数，实现精确控温和节能运行。

根据本发明的第一方面，提供一种智能温控系统，包括：

温度采集模块（10），用于实时采集环境温度数据；

控制模块（20），与所述温度采集模块（10）连接，用于根据所述环境温度数据和目标温度，采用自适应PID算法计算控制量；所述控制模块（20）包括：
　参数自适应单元（21），用于根据系统响应特性实时调整PID参数；
　预测控制单元（22），用于根据历史温度数据预测未来温度变化趋势；

执行模块（30），与所述控制模块（20）连接，用于根据所述控制量调节加热或制冷装置的输出功率。

根据本发明的一些实施例，所述参数自适应单元（21）基于神经网络的反向传播算法调整PID参数，所述神经网络以温度偏差和偏差变化率为输入，以PID参数调整量为输出。

根据本发明的一些实施例，所述预测控制单元（22）采用时间序列分析方法，建立温度变化的ARMA预测模型，预测未来1-5分钟的温度变化趋势。

根据本发明的一些实施例，所述控制模块（20）还包括节能优化单元（23），所述节能优化单元（23）用于根据峰谷电价时段和用户舒适度要求，优化温控策略。

本发明实施例的有益效果包括：

1. **控制精度高**：通过自适应PID算法，系统能够根据实时工况自动调整控制参数，将温度控制精度提高到±0.3℃。

2. **响应速度快**：结合预测控制，系统能够提前响应温度变化趋势，响应时间缩短30%以上。

3. **节能效果好**：通过节能优化策略，在保证舒适度的前提下，可降低能耗15-25%。

4. **适应性强**：无需人工整定参数，系统自动适应不同环境条件和负载变化。

实施例1

如图1所示，本实施例提供一种智能温控系统，应用于家用空调设备。该系统包括：

温度采集模块（10），包括设置在室内不同位置的多个温度传感器（11），用于实时采集环境温度数据。在本实施例中，温度传感器（11）采用数字温度传感器DS18B20，精度为±0.125℃，每30秒采集一次温度数据。

控制模块（20），与温度采集模块（10）通过I2C总线连接。控制模块（20）可以采用STM32系列微控制器，运行自适应PID控制程序。如图2所示，控制模块（20）包括：

参数自适应单元（21），用于根据系统响应特性实时调整PID参数。本实施例采用基于神经网络的自适应算法。神经网络采用3层BP网络，输入层有2个神经元（温度偏差e和偏差变化率de/dt），隐藏层有6个神经元，输出层有3个神经元（Kp、Ki、Kd的调整量ΔKp、ΔKi、ΔKd）。

神经网络训练采用在线学习方式，学习率η=0.3。PID参数的更新公式为：
Kp(k) = Kp(k-1) + ΔKp
Ki(k) = Ki(k-1) + ΔKi  
Kd(k) = Kd(k-1) + ΔKd

预测控制单元（22），用于根据历史温度数据预测未来温度变化趋势。本实施例采用ARMA(2,1)模型：
y(t) = φ1*y(t-1) + φ2*y(t-2) + ε(t) - θ1*ε(t-1)

其中，y(t)为t时刻的温度值，φ1、φ2为自回归系数，θ1为滑动平均系数，ε(t)为白噪声。模型参数通过最小二乘法在线辨识更新。

基于预测结果，控制系统可以提前调节输出功率，避免温度过冲。

执行模块（30），包括变频器（31）和压缩机（32）。控制模块（20）输出的控制量经DAC转换为0-10V模拟信号，控制变频器（31）调节压缩机（32）的转速，从而改变制冷/制热功率。

如图3所示，本实施例的自适应PID控制流程如下：

步骤S100：系统初始化，设定初始PID参数（Kp0=2.0，Ki0=0.5，Kd0=0.1）和目标温度Tset。

步骤S200：温度采集模块（10）采集当前温度T(k)，计算温度偏差e(k)=Tset-T(k)和偏差变化率de(k)=[e(k)-e(k-1)]/Δt。

步骤S300：将e(k)和de(k)输入神经网络，计算PID参数调整量ΔKp、ΔKi、ΔKd，更新PID参数。

步骤S400：基于ARMA模型预测未来温度变化，计算预测偏差ep。

步骤S500：根据公式u(k)=Kp*e(k)+Ki*∫e(k)dt+Kd*de(k)+Kf*ep计算控制量，其中Kf为预测补偿系数。

步骤S600：将控制量输出至执行模块（30），调节压缩机功率。

步骤S700：判断是否达到稳态，如未达到，返回步骤S200继续循环。

效果验证

为验证本发明的技术效果，进行了以下对比实验：

实验条件：室温25℃，目标温度设置为20℃，空调额定制冷功率3500W。

对照组：采用传统固定参数PID控制（Kp=2.0，Ki=0.5，Kd=0.1）
实验组：采用本发明的自适应PID控制

实验结果如图4所示：

1. 超调量：对照组为1.2℃，实验组为0.3℃，超调量降低75%。
2. 调节时间：对照组为12分钟，实验组为8分钟，响应时间缩短33%。
3. 稳态误差：对照组为±0.5℃，实验组为±0.2℃，精度提高60%。
4. 能耗：在24小时运行测试中，实验组比对照组节电18.5%。

实施例2

如图5所示，本实施例与实施例1的区别在于，控制模块（20）还包括节能优化单元（23）。

节能优化单元（23）根据峰谷电价时段和用户舒适度要求，优化温控策略。具体策略如下：

在峰电时段（电价较高），允许温度在目标温度±1℃范围内波动，降低压缩机运行频率，减少高电价时段的用电量。

在谷电时段（电价较低），提前进行预冷/预热，将温度控制在目标温度±0.3℃范围内，利用低电价时段储能。

根据用户历史数据学习用户的作息规律，在用户离家时段自动调高/调低设定温度，在用户回家前30分钟自动调整至舒适温度。

经实际测试，增加节能优化单元后，在保持用户舒适度的前提下，电费可进一步降低10-15%。
