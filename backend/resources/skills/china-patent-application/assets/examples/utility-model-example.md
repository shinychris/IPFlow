## 说明书

### 技术领域

本实用新型涉及照明装置技术领域，具体涉及一种折叠式便携台灯。

### 背景技术

台灯是日常生活中常用的照明设备，广泛应用于家庭、办公室、学校等场所。传统的台灯通常由底座、支架和灯头组成，体积较大，不便于携带和收纳。

现有的一些便携台灯虽然体积较小，但存在以下问题：

1. 结构简单，照明角度无法调节，使用不便。
2. 折叠结构不稳定，容易松动或损坏。
3. 电池容量小，续航时间短。
4. 功能单一，无法满足现代用户的需求。

因此，亟需一种结构紧凑、便于携带、功能丰富的台灯。

### 实用新型内容

本实用新型的目的在于提供一种折叠式便携台灯，以解决现有技术中存在的台灯体积大、不便携带的问题。

为实现上述目的，本实用新型采用如下技术方案：

一种折叠式便携台灯，包括：

底座（10），所述底座（10）内设有电池（11）和控制电路板（12）；

折叠支架（20），所述折叠支架（20）的下端与所述底座（10）铰接，所述折叠支架（20）包括至少两节可相对转动的支臂（21），相邻所述支臂（21）之间通过旋转关节（22）连接；

灯头（30），与所述折叠支架（20）的上端铰接，所述灯头（30）包括LED光源（31）和散热片（32）；

其中，所述旋转关节（22）包括：
    第一连接件（221），固定于一节所述支臂（21）的端部；
    第二连接件（222），固定于相邻所述支臂（21）的端部，与所述第一连接件（221）转动连接；
    阻尼结构（223），设置于所述第一连接件（221）和所述第二连接件（222）之间，用于提供转动阻力。

优选地，所述底座（10）上设有收纳槽（13），当所述折叠支架（20）完全折叠时，所述灯头（30）可嵌入所述收纳槽（13）内。

优选地，所述底座（10）的侧面设有充电接口（14）和USB输出接口（15），所述USB输出接口（15）用于为外部设备充电。

优选地，所述灯头（30）与所述折叠支架（20）的铰接轴垂直于所述支臂（21）之间的旋转轴，使所述灯头（30）可在两个相互垂直的平面内转动。

优选地，所述阻尼结构（223）包括：
    摩擦片（2231），固定于所述第一连接件（221）；
    弹性件（2232），设置于所述第二连接件（222），用于将所述摩擦片（2231）压紧。

本实用新型的有益效果：

1. 通过折叠支架设计，台灯可以折叠成平板状，体积小巧，便于携带和收纳。
2. 采用阻尼结构的旋转关节，可以在任意角度稳定支撑，调节灵活。
3. 灯头可双向转动，照明角度调节范围大。
4. 内置大容量电池，并可作为移动电源使用，功能实用。

### 具体实施方式

下面结合附图对本实用新型的具体实施方式进行详细说明。

实施例1

如图1至图3所示，本实施例提供一种折叠式便携台灯，包括底座（10）、折叠支架（20）和灯头（30）。

底座（10）呈扁平长方体状，由塑料材质制成。底座（10）内部设有电池（11）和控制电路板（12）。电池（11）采用5000mAh的锂离子聚合物电池，可为台灯提供长时间照明。控制电路板（12）上设有充电管理电路、LED驱动电路和USB输出电路。

如图5所示，底座（10）的上表面设有收纳槽（13），收纳槽（13）的形状和尺寸与灯头（30）相匹配。底座（10）的侧面设有充电接口（14）和USB输出接口（15）。充电接口（14）为Type-C接口，用于为电池（11）充电。USB输出接口（15）用于为手机等外部设备充电，使台灯具有移动电源功能。

折叠支架（20）包括三节支臂（21）：下支臂、中支臂和上支臂。下支臂的下端通过第一铰接轴与底座（10）的后端铰接，可相对于底座（10）在0-180度范围内转动。相邻支臂（21）之间通过旋转关节（22）连接。

如图4所示，旋转关节（22）包括第一连接件（221）、第二连接件（222）和阻尼结构（223）。第一连接件（221）固定于下支臂的上端，第二连接件（222）固定于中支臂的下端。第一连接件（221）和第二连接件（222）通过转轴转动连接。

阻尼结构（223）包括摩擦片（2231）和弹性件（2232）。摩擦片（2231）由耐磨材料制成，固定于第一连接件（221）的端面。弹性件（2232）为碟形弹簧，设置于第二连接件（222）内，对转轴施加轴向压力，将摩擦片（2231）压紧。通过调节弹性件（2232）的预紧力，可以调整旋转关节（22）的阻尼大小，使支臂（21）可以在任意角度稳定停留。

灯头（30）呈长条形，与上支臂的上端通过第二铰接轴铰接。第二铰接轴垂直于支臂（21）之间的旋转轴，使灯头（30）可在两个相互垂直的平面内转动，实现多角度照明调节。

灯头（30）内设有LED光源（31）和散热片（32）。LED光源（31）包括多个SMD LED灯珠，色温可调（3000K-6000K）。散热片（32）为铝合金材质，与LED光源（31）的基板紧密接触，将热量散发出去。

使用方法：

使用时，将折叠支架（20）从底座（10）上展开，根据需要的照明角度调节各节支臂（21）的相对位置和灯头（30）的角度。由于旋转关节（22）具有阻尼，支臂（21）可以在任意角度稳定支撑，无需额外的锁定机构。

携带时，将折叠支架（20）完全折叠，使灯头（30）嵌入底座（10）的收纳槽（13）内，台灯呈扁平状，厚度仅约3cm，便于放入背包携带。

实施例2

本实施例与实施例1的区别在于：折叠支架（20）包括两节支臂（21），结构更加简单。灯头（30）与上支臂固定连接，不可转动。这种结构适合对角度调节要求不高的场景，成本更低。

以上所述仅为本实用新型的较佳实施例而已，并不用以限制本实用新型，凡在本实用新型的精神和原则之内所作的任何修改、等同替换和改进等，均应包含在本实用新型的保护范围之内。

1. 一种折叠式便携台灯，其特征在于，包括：
    底座（10），所述底座（10）内设有电池（11）和控制电路板（12）；

    灯头（30），与所述折叠支架（20）的上端铰接，所述灯头（30）包括LED光源（31）；
    其中，所述旋转关节（22）包括第一连接件（221）、第二连接件（222）和阻尼结构（223），所述第一连接件（221）固定于一节所述支臂（21）的端部，所述第二连接件（222）固定于相邻所述支臂（21）的端部并与所述第一连接件（221）转动连接，所述阻尼结构（223）设置于所述第一连接件（221）和所述第二连接件（222）之间，用于提供转动阻力。

2. 根据权利要求1所述的折叠式便携台灯，其特征在于，所述底座（10）上设有收纳槽（13），当所述折叠支架（20）完全折叠时，所述灯头（30）可嵌入所述收纳槽（13）内。

3. 根据权利要求1所述的折叠式便携台灯，其特征在于，所述底座（10）的侧面设有充电接口（14）和USB输出接口（15），所述USB输出接口（15）用于为外部设备充电。

6. 根据权利要求1所述的折叠式便携台灯，其特征在于，所述阻尼结构（223）包括摩擦片（2231）和弹性件（2232），所述摩擦片（2231）固定于所述第一连接件（221），所述弹性件（2232）设置于所述第二连接件（222）并用于将所述摩擦片（2231）压紧。

7. 根据权利要求1所述的折叠式便携台灯，其特征在于，所述LED光源（31）包括多个SMD LED灯珠，色温可调。
