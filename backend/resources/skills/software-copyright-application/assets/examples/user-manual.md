# 第1章 软件概述

## 1.1 软件简介

食刻即选应用软件是一款面向日常生活消费领域的智能餐饮推荐小程序，旨在解决用户日常饮食选择的困扰。通过分析用户偏好、行为习惯和营养需求，为用户提供个性化的菜谱推荐和饮食计划管理功能。

本软件采用先进的混合推荐算法，结合内容过滤和协同过滤技术，实现实时用户行为学习和个性化权重调整，确保推荐结果既符合用户口味偏好，又能满足营养均衡需求。

软件基于微信小程序平台开发，用户无需下载安装，通过微信扫码或搜索即可使用，具有使用便捷、无需占用手机存储空间等优势。

## 1.2 主要功能

食刻即选应用软件主要包含以下功能模块：

| 功能模块 | 功能描述 |
|----------|----------|
| 智能推荐系统 | 基于AI算法的个性化菜谱推荐、用户偏好分析和学习、相似菜谱推荐、今日精选推荐 |
| 用户管理系统 | 微信小程序一键登录、用户偏好设置、用户行为追踪和分析、个人收藏和历史记录 |
| 菜谱管理功能 | 精选菜谱数据库、菜谱详情展示、菜谱搜索和筛选、菜谱收藏和评价 |
| 饮食计划管理 | 智能周计划生成、营养成分分析、计划调整和优化、健康饮食建议 |

# 第2章 系统安装与配置

## 2.1 服务端部署

### 2.1.1 环境准备

1. 安装Python 3.12
```bash
sudo apt update
sudo apt install python3.12 python3.12-pip
```

2. 安装PostgreSQL 15.16
```bash
sudo apt install postgresql-15
```

3. 安装Redis
```bash
sudo apt install redis-server
```

4. 安装Node.js 24.6
```bash
curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash -
sudo apt install -y nodejs
```

### 2.1.2 数据库初始化

1. 创建数据库
```bash
sudo -u postgres psql -c "CREATE DATABASE shikejixuan OWNER postgres;"
```

2. 执行初始化脚本
```bash
psql -U postgres -d shikejixuan -f server/migrations/001_initial_schema.sql
```

### 2.1.3 服务端配置

1. 安装Python依赖
```bash
cd server
pip install -r requirements.txt
```

2. 配置环境变量
```bash
export SECRET_KEY="your-secret-key"
export JWT_SECRET_KEY="your-jwt-secret-key"
export DATABASE_URL="postgresql://user:password@localhost:5432/shikejixuan"
export REDIS_URL="redis://localhost:6379/0"
export WECHAT_APPID="your-wechat-appid"
export WECHAT_SECRET="your-wechat-secret"
```

3. 启动服务
```bash
python server/app.py
```

## 2.2 客户端配置

### 2.2.1 小程序注册

1. 访问微信公众平台（https://mp.weixin.qq.com）
2. 注册小程序账号
3. 获取AppID和AppSecret
4. 配置服务器域名和业务域名

### 2.2.2 开发环境配置

1. 下载并安装微信开发者工具
2. 导入项目代码
3. 在project.config.json中配置appid
4. 修改config/api.js中的API地址

# 第3章 功能使用说明

## 3.1 用户登录

### 3.1.1 功能描述

用户通过微信小程序一键登录功能快速进入系统，无需单独注册账号。系统自动获取用户微信基本信息（昵称、头像等），并根据用户授权建立账号体系。

### 3.1.2 操作步骤

1. 打开微信，搜索"食刻即选"小程序
2. 点击搜索结果进入小程序
3. 首次进入时，系统弹出授权提示
4. 点击"允许"授权获取微信用户信息
5. 系统自动完成登录，进入首页

# 第4章 常见问题解答

## Q1：如何修改个人偏好设置？

A：点击底部导航"我的" → "偏好设置"，在设置页面修改后点击保存即可。

## Q2：为什么推荐的内容不符合我的口味？

A：请在"偏好设置"中完善您的菜系偏好、辣度接受程度等信息。系统会根据您的使用行为不断学习和优化推荐结果。

## Q3：如何分享菜谱给好友？

A：在菜谱详情页点击右上角的分享按钮，选择微信好友或生成分享卡片。

## Q4：收藏的菜谱在哪里查看？

A：点击底部导航"收藏"即可查看所有已收藏的菜谱。

## Q5：生成的饮食计划可以修改吗？

A：可以。在饮食计划页面，点击需要调整的餐点，可以更换为其他菜谱。

## Q6：软件需要付费使用吗？

A：食刻即选基础功能完全免费，部分高级功能可能需要开通会员。

## Q7：如何联系客服？

A：在"我的"页面点击"联系客服"，或发送邮件至 support@example.com。
