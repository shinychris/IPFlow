## [2.0.0] - 2024-01-15

### 🎉 重大更新

IPFlow v2.0 正式发布！这是一个里程碑版本，系统从单机应用升级为支持多租户的 SaaS 平台。

### ✨ 新增功能

#### 多租户与组织
- 完整的组织管理系统
- 多成员协作支持
- 基于角色的权限控制（RBAC）
- 组织邀请机制
- 组织级别资源隔离

#### 订阅与计费
- 分级订阅计划（免费版/专业版/企业版）
- 资源配额管理（项目数/存储空间/成员数）
- 限额控制服务
- 发票管理系统
- 支付提供商集成框架（预留 Stripe/支付宝/微信支付）

#### 管理后台
- 超级管理员仪表盘
- 用户管理（CRUD、搜索、过滤）
- 组织管理（审核、限额调整）
- 计划管理（动态配置）
- 审计日志查询

#### 审计与监控
- 完整的操作审计日志
- 请求追踪（X-Request-ID）
- 健康检查端点
- 系统指标收集
- 资源使用统计

### 🔧 技术改进

#### 后端
- 从 Express 迁移到 FastAPI
- 引入 SQLModel ORM
- 异步架构优化
- 多租户上下文管理
- 限流与安全防护

#### 前端
- 状态管理从 Redux 迁移到 Zustand
- TypeScript 类型系统完善
- 组件库升级
- 响应式设计优化

#### 数据库
- 新增订阅相关表
- 新增审计日志表
- 组织表结构优化
- 索引优化

### 📊 统计数据

- API 端点: 61个（新增30个）
- 测试用例: 120+（单元测试 + 集成测试）
- 代码行数: 后端 15,000+ 行，前端 25,000+ 行
- 文档页数: 50+ 页

### 📝 文档

- 全新的用户手册

## 版本说明

### 版本号格式

`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 修改
- **MINOR**: 向下兼容的功能新增
- **PATCH**: 向下兼容的问题修复

### 版本生命周期

| 版本 | 状态 | 支持时间 |
|------|------|----------|
| 2.0.x | 活跃开发 | 当前 |
| 1.5.x | 维护模式 | 2024-06-30 前 |
| 1.0.x | 已停止 | 不再支持 |

### 升级指南

#### 从 1.x 升级到 2.0

1. 备份现有数据
2. 更新代码到 v2.0 分支
3. 执行数据库迁移：`alembic upgrade head`
4. 更新环境变量配置
5. 重新部署应用

> ⚠️ 注意：v2.0 有重大架构变更，建议在测试环境验证后再升级生产环境。
